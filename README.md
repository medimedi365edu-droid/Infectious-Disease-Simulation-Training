# 🦠 감염병 모의훈련 시뮬레이션 (Infectious Disease Tabletop Exercise Simulator)

> 감염병 대응 역량 강화를 위한 **역할별 시나리오 기반 모의 훈련 웹 애플리케이션**

---

## 📌 프로젝트 개요

감염병 발생 상황을 단계별로 시뮬레이션하여, 담임교사·보건교사·영양교사·행정실·교장/교감·교육청·유관기관 등 각 역할이 실제 대응 절차를 훈련할 수 있도록 설계된 브라우저 기반 훈련 도구입니다.

**Firebase Realtime Database** 기반으로 실시간 멀티유저 동기화를 지원하며, 별도 설치 없이 **HTML 파일 하나**로 동작합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🎭 **다중 역할 지원** | 모더레이터(진행자), 담임교사, 보건(담당)교사, 영양(교)사, 행정실, 교장/교감, 교육(지원)청, 유관기관 |
| 📋 **다중 시나리오** | 인플루엔자, 수두, 결핵 등 학교 감염병별 시나리오 3단계 내장 |
| 🔥 **실시간 동기화** | Firebase Realtime Database로 모더레이터·참가자 간 실시간 상태 공유 |
| 📱 **QR 코드 초대** | 모더레이터 접속 시 QR 코드 자동 생성, 스마트폰으로 스캔하여 즉시 입장 |
| 🏠 **방 코드 시스템** | 시나리오·조 기반 방 코드 자동 생성, 참가자는 코드 입력으로 입장 |
| ⏱️ **단계별 타이머** | 각 훈련 단계마다 제한 시간 부여 및 긴급 알림 |
| 💬 **역할별 미션** | 각 역할에 맞는 상황 질문 및 모범 답안 제공 |
| 📊 **자동 피드백** | 훈련 완료 후 키워드 기반 답변 분석 및 점수 산출 |
| 🔧 **관리자 모드** | 시나리오 내용 수정·추가·삭제 가능 (비밀번호 보호) |
| 👥 **조 편성 지원** | 최대 10개 조 편성 및 조별 진행 |
| 🔤 **글씨 크기 조절** | 헤더 우측 가−/가+ 버튼으로 전체 콘텐츠 글씨 크기 조절, 기기에 저장 |
| 💾 **로컬 저장** | 커스텀 시나리오는 `localStorage`에 자동 저장 |

## 🆕 최신 업데이트 (2026-04-17)

### 👥 중간 입장 역할 선택 개선
- 1단계: 담임·보건 버튼만 표시 (혼란 최소화)
- 2단계 이후: 팝업으로 해당 단계의 역할 안내
- 신규 추가된 역할에 **NEW** 뱃지 표시

### 🔤 글씨 크기 조절 기능 추가
- 헤더 우측 상단 **가−** / **가+** 버튼으로 조절
- 상황·문제·키워드·피드백 전체 영역에 통합 적용
- 설정값 기기에 저장 (재접속 시 유지)

### 🔄 방 코드 초기화 버튼 추가
- 훈련 초기화 후 URL에 방 코드가 잔류하는 문제 수정
- 로그인 화면 하단에 초기화 버튼 추가
- 새로고침 없이 즉시 초기화 가능

### 📋 피드백·참여자 현황 글씨 크기 연동
- 다른 역할의 작성 현황 글씨 크기도 설정값에 연동
- 피드백 카드 내용·요약 글씨 크기 연동
- 모든 콘텐츠 영역 통합 적용

### 설치 없이 바로 실행

```bash
# 파일을 다운로드 후 브라우저에서 열기
open index.html
```

또는 파일을 더블 클릭하여 브라우저에서 직접 실행하세요.

> 오프라인 환경에서는 단독 실행(피드백·점수 확인)만 가능합니다.

### GitHub Pages로 배포

2. **Settings → Pages → Branch: main** 으로 설정합니다.

## 🎮 사용 방법

### 모더레이터 (진행자)

1. **시나리오 선택** — 홈 화면에서 훈련할 감염병 시나리오 클릭
2. **로그인** — 이름 입력, 조 번호 선택, **모더레이터(진행자)** 역할 선택 후 입장
3. **참가자 초대** — 자동으로 표시된 QR 코드를 화면에 띄우거나, URL·방 코드를 참가자에게 공유
4. **훈련 진행** — 단계 이동, 타이머 관리, 참가자 답변 실시간 확인
5. **피드백 확인** — 단계 종료 후 역할별 체크리스트 채점 결과 확인

> 📖 자세한 모더레이터 사용법은 [모더레이터 튜토리얼](모더레이터_튜토리얼.html)을 참고하세요.

### 훈련 참가자

1. **입장** — 모더레이터가 공유한 QR 코드 스캔 또는 방 코드 직접 입력
2. **역할 선택** — 이름 입력 후 본인 역할 선택 (조·시나리오 자동 설정)
3. **단계별 훈련** — 상황 설명 확인 → 역할별 미션 수행 → 답변 입력·제출
4. **피드백 확인** — 훈련 종료 후 모범 답안과 점수 확인

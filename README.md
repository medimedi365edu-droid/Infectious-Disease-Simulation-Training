# 🦠 감염병 모의 훈련 시뮬레이션 (Infectious Disease Tabletop Exercise Simulator)

> 감염병 대응 역량 강화를 위한 **역할별 시나리오 기반 모의 훈련 웹 애플리케이션**

---

## 📌 프로젝트 개요

이 프로젝트는 감염병 발생 상황을 단계별로 시뮬레이션하여, 보건 담당자·역학조사관·의료기관 등 각 역할이 실제 대응 절차를 훈련할 수 있도록 설계된 브라우저 기반 훈련 도구입니다.

별도의 서버나 설치 없이 **HTML 파일 하나만으로 동작**하며, 오프라인 환경에서도 사용 가능합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🎭 **다중 역할 지원** | 보건소장, 역학조사관, 의료기관, 사회복지사, 사회자(Moderator) 등 |
| 📋 **다중 시나리오** | 코로나19, 인플루엔자, 노로바이러스 등 감염병별 시나리오 내장 |
| ⏱️ **단계별 타이머** | 각 훈련 단계마다 제한 시간 부여 및 긴급 알림 |
| 💬 **역할별 미션** | 각 역할에 맞는 상황 질문 및 모범 답안 제공 |
| 📊 **자동 피드백** | 훈련 완료 후 키워드 기반 답변 분석 및 점수 산출 |
| 🔧 **관리자 모드** | 시나리오 내용 수정, 추가, 삭제 가능 (비밀번호 보호) |
| 👥 **조 편성 지원** | 최대 10개 조 편성 및 조별 진행 |
| 💾 **로컬 저장** | 커스텀 시나리오는 `localStorage`에 자동 저장 |

## 🚀 시작하기

### 설치 없이 바로 실행

```bash
# 파일을 다운로드 후 브라우저에서 열기
open index.html
```

또는 파일을 더블 클릭하여 브라우저에서 직접 실행하세요.

### GitHub Pages로 배포

1. 이 저장소를 Fork하거나 파일을 업로드합니다.
2. **Settings → Pages → Branch: main** 으로 설정합니다.
3. 생성된 URL로 접속하면 바로 사용 가능합니다.

## 🎮 사용 방법

### 훈련 참가자

1. **시나리오 선택** — 홈 화면에서 원하는 감염병 시나리오 클릭
2. **정보 입력** — 이름, 역할, 조 번호 선택 후 입장
3. **단계별 훈련** — 상황 설명 확인 → 역할별 미션 수행 → 답변 입력
4. **피드백 확인** — 훈련 종료 후 모범 답안과 점수 확인

### 사회자 (Moderator)

- 전체 진행 제어 (단계 이동, 타이머 관리)
- 참가자 답변 실시간 확인

### 관리자

1. 홈 화면 하단 **관리자 모드** 버튼 클릭
2. 비밀번호 입력 (기본값: `admin1234`)
3. 시나리오 내용 편집, 새 시나리오 추가/삭제
4. 관리자 비밀번호 변경 가능
